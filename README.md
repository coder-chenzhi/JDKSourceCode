# JDKSourceCode

## English

## Introducion
Read Oracle JDK (1.8.0_65) source code

- The structure of this project is the same with the JDK source code.
- The notes are in markdown files which are the same name with the corresponding JDK java file.
- This project will store some code to assist to understand JDK source code.
